# **YOUR VOICEPACK NAME** Voicepack
A Lethal Company Mod made for Lethal Bots to add **VOICE NAME** as a voice profile for the bots to use.</br>
Everyone must have the mod in order for it to work!<br/>

## Installation
1. Download the latest release from Thunderstore.io using your favorite mod manager (e.g., Thunderstore Mod Manager, R2ModMan, Gale, etc.).
2. Make sure to have Lethal Bots installed as this mod does nothing without it.
3. Set the bot's voice to **FOLDER NAME** (ex. Francis_L4D1) to tell the bot to use the voicepack
